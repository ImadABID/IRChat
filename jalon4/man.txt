

Nickname

    /nick <you_name> : Choose nickname

Messages :

    /who                                    : List connected users.
    /whois <nickname>                       : List <nickname>'s infos.
    /msg <reciver_nickname> <the message .> : Send a message to <reciver_nickname>.
    /msgall <the message .>                 : Send a message to all users.

Channels :
    
    /channel_list           : Get channel list.
    /create <channel_name>  : Create a channel.
    /join <channel_name>    : Join a channel.
    /quit <channel_name>    : Quit a channel.

Files :

    /file_hist                              : Display interactively files transfer history & progress.
    /send <file_name> <reciver_nickname>    : Send file.
    /file_accept <file_name>                : Accept a requested file. /file_hist can help to find the <file_name>.
    /file_reject <file_name>                : Reject a requested file. /file_hist can help to find the <file_name>.

Quit :

    /quit : Quit the prog
